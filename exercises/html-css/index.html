Vim Text Objects Guide

Published on January 1, 2024

Text objects in vim allow you to quickly select and manipulate content within various delimiters like tags, quotes, and brackets.

Common Text Objects:

  • it - inner tag content
  • at - around tag (including tags)
  • i" - inner quotes
  • a" - around quotes
